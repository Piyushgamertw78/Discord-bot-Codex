 🤖 Discord Bot

Discord bot with 130+ commands featuring AI chat, economy system, moderation tools, and more!

## ✨ Features
- 💰 Economy System
- 📊 Leveling System
- 🛡️ Moderation Tools
- 🎮 Fun Commands
- 🎉 Giveaway System

## 🚀 Commands

Use `!help` to see all available commands

## 🔧 Setup

1. Clone repository
2. Install dependencies: `npm install`
3. Set environment variables:
   - `TOKEN` - Bot token
   - `PREFIX` - Command prefix
   - `OWNER_ID` - Owner Discord ID
4. Run: `npm start`
